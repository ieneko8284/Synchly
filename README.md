# Synchly (シンクリー)

「感性の同期」をコンセプトにした、価値観重視のマッチングアプリケーションです。
学校の卒業制作として、PHPとJavaScript、MySQLを使用してフルスタックで開発しています。

## 🌟 プロジェクトの概要
一般的なマッチングアプリとは異なり、共通の趣味や感性（音楽、文学、プログラミング等）をベースに、より深い繋がりを提供することを目指しています。

## 🚀 主な機能
- **ユーザー認証**: UUIDを使用したセキュアな会員登録・ログイン機能
- **マッチングシステム**: 異性ユーザーのレコメンド、相互いいねによるマッチング成立
- **リアルタイム通知**: マッチング成立時の即時通知（Ajaxによるポーリング）
- **チャット機能**: マッチング相手とのリアルタイムなメッセージ交換
- **マイページ管理**: 自己紹介やプロフィールの編集（実装予定）、退会機能（物理削除）

## 🛠 使用技術
### バックエンド
- **PHP 8.x**: 
  - フロントコントローラーパターンによるルーティング
  - Repositoryパターンによるデータアクセスの分離
  - ComposerによるPSR-4準拠のオートロード
- **MySQL 8.0**: 
  - UUID(v4)を主キーに採用
  - 外部キー制約（ON DELETE CASCADE）による整合性保持

### フロントエンド
- **Vanilla JavaScript**: 
  - `async / await` を活用した非同期通信
  - DOM操作によるSPA（Single Page Application）ライクな画面遷移
  - `window` オブジェクトへのグローバル展開によるイベントハンドリング

### ツール
- **Composer**: 依存関係管理
- **Git / GitHub**: バージョン管理

## 📂 ディレクトリ構造
```text

├── public/         # ドキュメントルート（index.php, CSS, 画像）
